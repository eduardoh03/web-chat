# Chat 

**Chat** is a modern, real-time interactive chat application built with Node.js, Express, and Socket.io. It features a sleek dark-themed UI, user identification, and real-time typing indicators.

## Features

- **Real-time Messaging**: Instant message delivery using WebSocket technology.
- **Modern UI/UX**: A responsive, dark-themed interface with glassmorphism effects.
- **User Identification**: Users set a nickname upon joining.
- **Typing Indicators**: See when others are typing in real-time.
- **System Notifications**: Alerts when users join or leave the chat.
- **Responsive Design**: Works seamlessly on desktop and mobile devices.

## Technologies

- **Backend**: Node.js, Express
- **Real-time Engine**: Socket.io
- **Frontend**: HTML5, CSS3 (Variables, Flexbox), Vanilla JavaScript

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/eduardoh03/web-chat.git
   cd web-chat
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the server**
   ```bash
   npm start
   # or
   node server.js
   ```

4. **Access the application**
   Open your browser and navigate to `http://localhost:3000`.
